# Allo Série

Gestionnaire de séries.

## Installation

```
git clone https://github.com/dardinier/AlloSerie
cd AlloSerie
npm install
npm run startDev
```

* Ouvrir un navigateur et aller sur : [http://localhost:4000](http://localhost:4000)

## Commandes NPM

* npm run build
* npm run watch
* npm run start
* npm run startDev
* npm run testServer
